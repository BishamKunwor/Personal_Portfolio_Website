Inspired Page

While listening music on spotify web, going into different tabs; I saw hover effects on navigation links. These hover effect were efficient way to make an illusion of a buttion in something like the text. The text with hover effect could be made to appear as if it was an actual button.

My Page

The concept of this navigaiton link hover effect was considered and used in this website by giving the text some effects. Whenever the user hover over the navigation link, a white button would appear beneath the text making an illusion of a digital buttion being present. Each button was colored white to match the site design.
