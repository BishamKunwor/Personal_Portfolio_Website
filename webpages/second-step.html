Inspired Page

When one of my friend sent me his progress in mockplus website design, at the loading page I saw a box which triggered the concept of the box website and all content to be wrapped to the box. The box must be curved around the edge and it must contain shadow to give it a 3d effect that it is floating above the webpage.

My Page

The main theme of my site is box design. This means all data that is represented or all the content in the website should be wrapped inside a box. The box design should be seperate for seperate type of information. It must be attractive to the user so some transitions or animation is needed to make the website more intractive.
